Absolute Hyperlink
An Absolute Hyperlink is a link to an absolute location on the web. You use this type of hyperlink when you need to connect the user to a URL that resides on the web and is not in your site folder. An example: Click here to visit Google
Relative Hyperlink
A Relative Hyperlink is a link to a webpage or media element within your site folder. An example of linking to a file within my main site folder is Click here to visit my lists page.
The following resources also explain how to create links. Please look at them for better understanding:
